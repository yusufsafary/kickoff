# Penalty Kick Game

Game tendangan pinalti sepak bola yang realistis dan seru!

## Fitur
- Stadion malam hari dengan penonton dan lampu sorot
- Fisika bola realistis dengan kurva dan spin
- Kiper AI dengan animasi diving
- Sistem skor 5 tendangan
- Efek visual ledakan saat gol

## Stack
- React + Vite + TypeScript
- HTML5 Canvas (rendering game)
- Tailwind CSS

## Cara Main
1. Klik **MULAI MAIN**
2. Arahkan kursor ke area gawang
3. Klik untuk menendang
4. Selesaikan 5 tendangan

## Run Lokal
```bash
pnpm install
pnpm --filter @workspace/penalty-kick run dev
```
